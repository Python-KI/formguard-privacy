FormGuard Privacy Policy / プライバシーポリシー

Last updated: 2026-06-12

English

FormGuard stores your data locally. It never sends your text anywhere.

What FormGuard stores

  • Text you type into form fields (textareas, text inputs, rich-text editors), together with the page's origin and path, the field's label, and a timestamp.
  • Your settings (auto-save on/off, excluded sites, license status).

All of this is stored only inside your browser (IndexedDB and chrome.storage.local in the extension's own storage area). FormGuard has no server and performs no analytics or tracking.

What FormGuard never stores

  • Password fields (type="password")
  • Credit card fields (autocomplete="cc-*"), one-time codes, hidden fields
  • Anything typed in Incognito windows
  • URL query strings or fragments (only origin + path are kept)

Network communication

The only network communication FormGuard performs is with ExtensionPay (extensionpay.com), the payment provider used for the optional Premium upgrade and free trial. This is used solely to check license status and process payments. Your saved text is never included in that communication. See ExtensionPay's privacy policy for details.

Data deletion

Use the "Delete all data" button in the popup to permanently erase all saved text at any time. Uninstalling the extension also removes all locally stored data.

日本語

FormGuardはデータをローカルにのみ保存します。入力内容を外部に送信することは一切ありません。

保存するもの

  • フォーム欄に入力されたテキスト(ページのオリジンとパス、欄のラベル、日時とともに保存)
  • 設定(自動保存のオン/オフ、除外サイト、ライセンス状態)

これらはすべてブラウザ内(拡張機能専用のIndexedDBおよびchrome.storage.local)にのみ保存されます。FormGuardにサーバーはなく、アクセス解析・トラッキングも行いません。

保存しないもの

  • パスワード欄(type="password")
  • クレジットカード欄(autocomplete="cc-*")、ワンタイムコード、hidden欄
  • シークレットウィンドウでの入力
  • URLのクエリ文字列・フラグメント(保存するのはオリジンとパスのみ)

通信について

FormGuardが行う唯一の通信は、プレミアム版の決済・無料トライアルに使用する ExtensionPay(extensionpay.com) とのライセンス確認・決済処理です。この通信に入力テキストが含まれることはありません。詳細は ExtensionPayのプライバシーポリシーをご覧ください。

データの削除

ポップアップの「全データを削除」ボタンで、保存されたテキストをいつでも完全に消去できます。拡張機能をアンインストールした場合も、ローカルに保存されたデータはすべて削除されます。
